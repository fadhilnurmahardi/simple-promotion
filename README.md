# simple-promotion

Service to handle promotion product with different type of promotion. Can be call use HTTP or Graphql (See API docs below.)

## Download Depedency

```go mod download```

## How to Run

```go run cmd/app/main.go```

## How to Test

```go test ./... -coverprofile cover.out```
